# King of Time Travel
A Unity‑built 2D adventure platformer where you play as a time‑traveling King, exploring medieval castle rooms and futuristic space‑station chambers. Collect keystones, unlock doors, defeat enemies with your royal hammer, and avoid deadly hazards as you search for the hidden treasure.

## Gameplay Overview🚀
Travel through interconnected rooms that blend:
Medieval castle interiors
Futuristic space‑station corridors
Animated transitions between time periods

As the King, your mission is to:
Collect keystones to unlock doors
Jump on enemies using your hammer attack
Avoid hazards like green burning liquid
Navigate platforming challenges across two different eras

## Controls 🎮
Arrow Keys — Move left and right
Up Arrow — Enter doors / interact
Spacebar — Jump
Hammer Jump — Jump onto enemies to defeat them

## Features ✨
2D platformer movement
Time‑travel themed level transitions
Keystone + door unlocking system
Enemy defeat using hammer jump
Environmental hazards and traps
Castle + sci‑fi hybrid level design
Built with Unity and C# scripting
